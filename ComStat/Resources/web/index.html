Valid Commands
General commands:
  disk:
    get disk information
  smbios:
    get system management BIOS information
  gpu:
    get gpu information
  cpu:
    get cpu information
  network:
    get network information
  system:
    get system information
  physicalmemory:
    get physical memory information
  registry:
    get hardware id information
Custom commands:
  [general command] get [characteristic name, ...]:
    get characteristic information
Other commands:
  all:
    get all information (execute all commands)
  real time:
    get cpu and memory logs in real time
    (also it save log in logs/log.csv)
  music on:
    music on
  music off:
    music off
  save:
    save all statistics in logs/statistics.csv
  help:
    watch valid commands
  exit:
    exit from application
